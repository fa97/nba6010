2a. Write a sample query relevant to your business, and save it in your database. Describe your query on your a2 page by writing your query in SQL, and in English

We created the following query:

SELECT * FROM Customer WHERE (((Customer.[state])='NY'));

The above query selects all customer whose residential state is NY. It fetches all the columns of the Customer table which is indicated by '*'. If we want only specific columns we can specify the column names instead of *. The * specifies that all columns of the table be included in the output.
